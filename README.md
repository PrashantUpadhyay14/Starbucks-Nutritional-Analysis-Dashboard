# ☕ Starbucks Nutritional Analysis Dashboard | Power BI

## 📌 Project Overview

This Power BI dashboard provides a comprehensive nutritional analysis of Starbucks beverages. It helps analyze beverage categories, calorie distribution, caffeine content, protein levels, sugar content, and beverage preparation methods through interactive visualizations and KPI tracking.

The dashboard enables users to compare nutritional values across beverage categories and identify healthier as well as high-energy drink options.

---

# 🎯 Business Objective

The objective of this dashboard is to transform Starbucks beverage nutritional data into meaningful insights that support menu analysis, customer awareness, and product comparison through interactive reporting.

---

# 📊 Dashboard KPIs

- ☕ Total Beverage Categories
- 🍬 Average Sugar
- 🔥 Average Calories
- 🥛 Average Protein

---

# 📈 Analysis Performed

## 1️⃣ Beverage Overview

- Total beverage categories available
- Nutritional summary across all beverages

---

## 2️⃣ Calories Analysis

- Average calories by beverage category

---

## 3️⃣ Beverage Category Distribution

- Distribution of Starbucks beverage categories

---

## 4️⃣ Global Beverage Comparison

- Comparison of beverage categories based on nutritional metrics

---

## 5️⃣ Protein Analysis

- Average protein by beverage
- Highest protein beverages
- Category-wise protein comparison

---

## 6️⃣ Caffeine Analysis

Top 5 beverages with the highest caffeine content.

This analysis helps identify beverages that provide the highest caffeine levels.

---

## 7️⃣ Beverage Preparation Analysis

Interactive comparison based on beverage preparation types.

Users can dynamically filter beverage categories using:

- Beverage Preparation
- Protein Range

---

# 💡 Key Business Insights

- The dashboard analyzes **33 Starbucks beverage categories**.
- Smoothies have the highest average calorie content among beverage categories.
- White Chocolate beverages provide the highest average protein levels.
- Signature Espresso beverages are among the highest calorie coffee drinks.
- The dashboard identifies the Top 5 beverages with the highest caffeine content.
- Beverage preparation filters allow dynamic nutritional comparisons.
- Protein range filtering enables users to identify healthier beverage options.
